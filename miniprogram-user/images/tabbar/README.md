# TabBar图标说明

需要准备以下图标文件（PNG格式，81x81像素）：

## 首页
- `images/tabbar/home.png` - 首页图标（未选中）
- `images/tabbar/home-active.png` - 首页图标（选中）

## 购物车
- `images/tabbar/cart.png` - 购物车图标（未选中）
- `images/tabbar/cart-active.png` - 购物车图标（选中）

## 消息
- `images/tabbar/message.png` - 消息图标（未选中）
- `images/tabbar/message-active.png` - 消息图标（选中）

## 我的
- `images/tabbar/profile.png` - 我的图标（未选中）
- `images/tabbar/profile-active.png` - 我的图标（选中）

## 其他图标
- `images/default-avatar.png` - 默认头像（160x160像素）
- `images/default-dish.png` - 默认菜品图片（200x200像素）
- `images/banner1.png` - Banner图片1（750x300像素）
- `images/banner2.png` - Banner图片2（750x300像素）
- `images/banner3.png` - Banner图片3（750x300像素）

## 图标设计建议
- 使用美团风格的图标
- 未选中状态：灰色 (#86909C)
- 选中状态：红色 (#FF4D4F)
- 线条简洁，易于识别
